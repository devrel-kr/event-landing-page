AI 코딩 어시스턴트와 함께하는 실전 개발 워크샵

📅 2026년 4월 25일 (토)

🕐 12:00 - 18:00

📍 한국 마이크로소프트 13층

서울 종로구 종로1길 50, 더케이트윈타워 A동

GitHub Copilot Dev Days는 전 세계에서 동시에 열리는 커뮤니티 주도의 글로벌 이벤트입니다. GitHub Copilot의 강력한 AI 코딩 기능을 직접 체험하고, 실전 워크샵을 통해 AI 기반 개발 워크플로우를 함께 배웁니다. 서울에서는 오는 4월 25일(토) 광화문 한국 마이크로소프트 오피스에서 열립니다.

🎯 이런 분들을 위한 행사입니다

🔰

AI 코딩을 처음 시작하는 분

GitHub Copilot을 처음 사용해보시는 분도 환영합니다. AI 보조 개발의 기초부터 시작해 실전에 바로 활용할 수 있는 스킬을 익힐 수 있습니다.

⚡

업무 효율을 높이고 싶은 개발자

이미 Copilot을 사용하고 계신 분이라면 최신 기능과 고급 팁을 통해 개발 워크플로우를 한 단계 업그레이드할 수 있습니다.

🤝

커뮤니티와 연결되고 싶은 분

서울의 개발자 커뮤니티와 함께 네트워킹하고, 전 세계 동시 진행 이벤트의 일원으로 함께해 보세요.

GitHub Copilot 워크샵
실전 AI 보조 개발 체험

GitHub Copilot을 활용한 애플리케이션 설계 및 개발 워크샵을 진행합니다. 실제 코드를 작성하면서 AI 보조 개발의 효율을 직접 체험해 보세요. 세부 내용은 추후 안내 예정입니다.

  • GitHub Copilot 소개 및 핵심 기능 시연
  • VS Code, Visual Studio 등 에디터 통합 실습
  • GitHub Copilot CLI 활용
  • AI 기반 코드 작성·리뷰·개선 실습
  • 로컬 커뮤니티 발표 세션
